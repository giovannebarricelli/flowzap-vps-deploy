# FlowZap - Luxury Gold Edition 🚀

Painel frontal exclusivo para automação de marketing via WhatsApp, focado em alta performance e segurança (Anti-Ban).

## 💎 Diferenciais
- **Interface Luxury:** Design roxo imperial com detalhes em ouro champagne.
- **Segurança:** Trava mínima de 2.5 min por campanha e pausas automáticas de descanso.
- **Inteligência:** Higienização automática de números para o padrão internacional (55).
- **Logs em Tempo Real:** Painel lateral para monitoramento de sucessos e falhas.
- **Controle Total:** Botões de Pausar e Cancelar operação em tempo real.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Custom Luxury Design)
- JavaScript Vanilla (Sem dependências externas)

## 📖 Como Rodar
1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
3. Para acessar o painel em modo de demonstração, utilize a instância: **teste1**.
